Quantum Route Optimization

Solve multi-city routing problems using a QAOA-inspired quantum circuit. Feed in a distance matrix and get the shortest route back — no classical solver required.

What is QAOA? The Quantum Approximate Optimization Algorithm encodes the routing problem as a cost Hamiltonian and uses quantum interference to find low-cost solutions. It's particularly powerful for combinatorial problems like the Travelling Salesman Problem that are computationally expensive classically.
Route Optimization
Provide a square NxN distance matrix where distances[i][j] is the distance from city i to city j. The diagonal must be 0. The API returns the optimal visit order and total distance.

The default example below has 4 cities. You can expand it to any size.
POST /quantum/logistics/optimize
—
How to read the response
The response contains two fields:

route — an ordered list of city indices representing the optimal visit sequence. For example [0, 1, 3, 2] means: start at city 0, go to city 1, then city 3, then city 2.

total_distance — the sum of distances along the optimal route.

Example response:
{ "route": [0, 1, 3, 2], "total_distance": 80.0 }
Use cases: delivery fleet routing, warehouse pick paths, network packet routing, supply chain optimization.
