Quantum Portfolio Optimization

Balance expected returns against risk using a VQE-inspired quantum ansatz. Select the optimal subset of assets for any risk tolerance — from conservative to aggressive.

What is VQE? The Variational Quantum Eigensolver uses a parameterized quantum circuit (ansatz) to find the minimum energy state of a problem Hamiltonian. Applied to finance, it finds the asset allocation that minimizes risk-adjusted cost — equivalent to solving the Markowitz portfolio optimization problem.
Portfolio Optimization
Provide expected returns per asset, an NxN covariance (risk) matrix, and a risk tolerance between 0.0 (most conservative) and 1.0 (most aggressive). The API returns the optimal allocation and selected assets.
POST /quantum/finance/portfolio
—
How to read the response
The response contains two fields:

allocation — a map of each asset to its weight in the portfolio. A weight of 0.5 means 50% of the portfolio, 0.0 means the asset was excluded.

selected_assets — the list of assets that were included in the optimal portfolio.

Example response:
{ "allocation": { "asset_0": 0.5, "asset_1": 0.0, "asset_2": 0.5, "asset_3": 0.0 }, "selected_assets": ["asset_0", "asset_2"] }
Use cases: robo-advisors, hedge fund allocation, pension fund rebalancing, crypto portfolio management.
